# DOM Manipulation with Javascript - Trilha dev

Praticando as aulas da Trilha dev com HTML, CSS e Javascript

O seguinte é um projeto de prática da Trilha-dev do curso Floripa Mais Tec onde criei um arquivo em HTML e CSS com classes e ID's e depois criei um arquivo em Javascript para poder manipular o DOM e criei vários prompts com os que o usuário pode acessar ou criar uma conta.

**English:**

The following is a practice project from the Dev Track of the Floripa Mais Tec course where I created an HTML and CSS file with classes and IDs. Then, I created a JavaScript file to manipulate the DOM and added various prompts that the user can access or use to create an account.

**Español:**

Este es un proyecto de práctica de la Trilha-dev del curso Floripa Mais Tec, donde creé un archivo en HTML y CSS con clases e IDs. Luego, creé un archivo en JavaScript para manipular el DOM y añadí varios prompts con los que el usuario puede acceder o utilizar para crear una cuenta.
